# SCT_CS_2

Features:

*Fully interactive GUI interface using Tkinter.
*Browse and select images externally.
*Choose Encrypt or Decrypt action.
*Supports multiple encryption/decryption methods: xor, add, swap.
*Enter a key for applicable methods (xor and add).
*Automatically saves results as encrypted.png or decrypted.png.
*Lightweight and fast.

Technologies Used:

*Python 3
*Tkinter (GUI)
*Pillow (Image Processing)
*NumPy (Array Manipulation)

Usage:

*Click Browse to select an image.
*Choose Encrypt or Decrypt from the dropdown.
*Select the Method (xor, add, swap).
*Enter a key if using xor or add (default: 123).
*Click Run → the processed image is saved automatically.
*Output files: encrypted.png or decrypted.png.

Workflow:

*Image selection → Action selection → Method selection → Key input → Run → Result saved
*Learning Outcome
*Practical experience in GUI development using Tkinter.
*Understanding image processing and pixel-level encryption.
*Implementing basic cryptography techniques in Python.
